Tags de Cabeçalho

  • h1

    - Header 1, geralmente usado como título principal de uma página ou seção, pois possui a maior fonte e maior destaque em relação as outras tags de cabeçalho

  • h2

    - Header 2, pode ser usado como título principal ou subtítulo de uma página ou seção

  • h3

    - Header 3, é o terceiro nível de cabeçalho

  • h4

    - Header 4, é o quarto nível de cabeçalho

  • h5

    - Header 5, é o quinto nível de cabeçalho

  • h6

    - Header 6, é o sexto nível de cabeçalho

Párágrados - tag p

Tag utilizada para ser aplicada em parágrafos

Destacar textos - tag mark

Tag utilizada para destacar partes de um texto

Fontes pequenas - tag small

Tag utilizada para diminuir o tamanho de um texto

Texto sublinhados - tag u

Textos Negritoa - tag strong

Tag utilizada para deixar um texto em negrito

Listas

Listas servem para colocar itens em sequência, sejam elas ordenadas(ol) ou não ordenadas(ul), os itens serão representados pela tag li, que virão dentro das tags principais

Listas Ordenadas - ol

Tag responsável por exibir listas ordenadas, como a abaixo

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas - ul

Tag responsável por exibir listas não ordenadas, como a abaixo

  • Item 1
  • Item 2
  • Item 3

Âncoras

Tag responsável por criar um link, que pode redirecionar para uma página externa ou do mesmo site

Os principais atributos são o href, onde deverá ser informada a url para a qual o usuário será redirecionado ao clicar no link, e o target, que definirá onde o novo link será aberto, podendo receber os seguintes valores:

  • _self = abre o link na mesma aba
  • _self = abre o link em uma aba diferente

Linha Horizontal - tag hr

Tag responsável por traçar uma linha Horizontal, tendo um valor temático, como por exemplo para separar seções de uma história, como as que separam as seções deste guia

Texto acima - tag sup

Citações - tag blockquote

Tag responsável por apresentar citações.

Aqui poderia ser uma citação, referência, etc, como um trecho de um texto de um livro por exemplo.

Este elemento é utilizado para representar uma parte do texto que foi excluída de um documento. São representados desta forma
